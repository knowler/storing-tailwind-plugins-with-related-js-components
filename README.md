# Proof of Concept: storing Tailwind plugins with JS components

The thought was to store the corresponding Tailwind plugin with
the JS component like this:

```
src/
└── components/
   ├── Cluster
   │  ├── Cluster.js
   │  └── Cluster.tailwind.js
   └── Stack
      ├── Stack.js
      └── Stack.tailwind.js
```
